- תהליכי בריאה ותיקון (מאמרות הבריאה, ניסיונות אברהם)

### 📁 תיקיית "13" - שלוש עשרה שערי חכמה

- מידות ותיקונים (מידות הרחמים, תיקוני דיקנא)
