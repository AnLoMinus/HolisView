## 📝 הוראות שימוש

1. בחר את שיטת הניתוח המתאימה לטקסט שלך
2. עקוב אחר המבנה המוצע בכל שיטה
3. השתמש בפורמט התשובה המובנה
4. שים לב לדגשים המיוחדים בכל שיטה

## 🎯 מטרות הניתוח

- הבנה מעמיקה של טקסטים
- חיבור בין רבדים שונים של משמעות
- יישום מעשי של התובנות
- פיתוח חשיבה ביקורתית ויצירתית

## 💡 טיפים לניתוח מוצלח

1. שמור על הפרדה ברורה בין הרבדים השונים
2. השתמש בשאלות מנחות להעמקה
3. חבר את הניתוח לידע קיים
4. הצע כיווני מחשבה נוספים

## 📚 מקורות נוספים

- ספרי יסוד בקבלה
- כתבים פילוסופיים
- מקורות מודרניים רלוונטיים
- מאמרים מקצועיים
