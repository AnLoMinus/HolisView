# 🌱 תיקון המידות ועבודת המוסר

## מבוא לעבודת המידות

תיקון המידות הוא חלק מרכזי בעבודת שערי האורה. כל שער מאיר מידה מסוימת ומאפשר את תיקונה. העבודה על המידות היא הבסיס לקבלת האור העליון.

## 🔍 זיהוי המידות

### מידות היסוד

1. **אהבה** - חסד והשפעה לזולת
2. **יראה** - גבורה וגבולות
3. **רחמים** - תפארת ואיזון
4. **בטחון** - נצח ועמידה איתנה
5. **הודאה** - הוד וביטול
6. **התקשרות** - יסוד וחיבור
7. **שפלות** - מלכות וקבלה

### מפת המידות האישית

- **מידות חזקות** שיש לכוון
- **מידות חלשות** שיש לחזק
- **מידות מעורבות** שיש לאזן
