# 💭 שאלות להתבוננות ותרגול

## שאלות יומיות

### 🌅 בוקר

1. **מהו השער שאני עובד עליו היום?**
2. **איזו מידה אני צריך לתקן דרכו?**
3. **מה המטרה שלי להיום בעבודת המידות?**
4. **איך אני מתכוון להתחבר לשם ה' של היום?**

### 🌞 במהלך היום

1. **האם אני מרגיש את האור של השער?**
2. **איך אני מתמודד עם הקליפה של היום?**
3. **מה הניסיונות שעולים בקשר לשער?**
4. **איך אני יכול להתחזק ברגע זה?**

### 🌙 ערב

1. **מה למדתי היום על עצמי?**
2. **איזה תיקון הצלחתי לעשות?**
3. **מה אני לוקח איתי למחר?**
4. **איך התקדמתי בעבודת המידות?**

## שאלות שבועיות

### 📝 סיכום שבועי

1. **איזה שערים עברתי השבוע?**
2. **מה היה האתגר המרכזי?**
3. **איזו התקדמות חשתי?**
4. **מה אני רוצה לשפר בשבוע הבא?**

### 🎯 תכנון קדימה

1. **מה המטרה שלי לשבוע הבא?**
2. **איך אני מתכונן לשערים הבאים?**
3. **איזה כלים אני צריך?**
4. **מי יכול לעזור לי בדרך?**

## שאלות חודשיות

### 📊 בדיקת התקדמות

1. **איך התקדמתי החודש בעבודת השערים?**
2. **אילו שינויים אני רואה במידות שלי?**
3. **מה השתנה בחיבור שלי לתפילה?**
4. **איך השתפר הלימוד שלי?**

### 🌱 צמיחה והתפתחות

1. **מה גיליתי על עצמי?**
2. **איזה כוחות חדשים מצאתי?**
3. **איך אני מרגיש את ההתקדמות הרוחנית?**
4. **מה המטרה שלי לחודש הבא?**

## שאלות לעבודה עם שערים ספציפיים

### 🔍 זיהוי הקליפה

1. **איך הקליפה מתבטאת בחיי?**
2. **מתי אני נופל בה?**
3. **מה מעורר אותה?**
4. **איך אני יכול להתגבר?**

### ✨ חיבור לאור

1. **איך אני מרגיש את האור של השער?**
2. **מה עוזר לי להתחבר אליו?**
3. **איך אני יכול להגביר את האור?**
4. **איך אני משתף אחרים באור?**

## 📝 הנחיות לשימוש בשאלות

1. **בחר זמן קבוע** ליומן התבוננות
2. **כתוב בכנות** את התשובות
3. **חזור ובדוק** את ההתקדמות
4. **שתף עם חברותא** או מדריך רוחני
