# 🌅 הכנה לגאולה ותיקון העולם

## מבוא לעבודת הגאולה

עבודת שערי האורה היא הכנה לגאולה השלמה. כל שער שנפתח מקרב את העולם לתיקונו, וכל אור שמתגלה מאיר את דרך הגאולה.

## 🌟 שלבי הגאולה

### תהליך הגאולה

1. **התעוררות מלמטה**
   - עבודה אישית
   - תיקון המידות
   - הפצת אור

2. **התעוררות מלמעלה**
   - גילוי אורות חדשים
   - ניסים ונפלאות
   - שינוי המציאות

3. **חיבור שמים וארץ**
   - אחדות מושלמת
   - גילוי אלוקות
   - תיקון העולם

## 💫 תיקון העולם

### רמות התיקון

2. **תיקון הכלל**
   - אחדות ישראל
   - תיקון החברה
   - הפצת תורה

3. **תיקון העולם**
