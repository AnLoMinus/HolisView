# 📚 לימוד תורה וקבלה

## מבוא ללימוד הקבלה

לימוד תורת הקבלה הוא חלק מרכזי בעבודת שערי האורה. הלימוד מאפשר הבנה עמוקה של השערים, המלאכות והאורות, ומהווה כלי מרכזי בתהליך התיקון.

## 🎯 מטרות הלימוד

### מטרות כלליות

1. **הבנת מבנה העולמות**
2. **הכרת שמות הקודש**
3. **תפיסת תהליכי התיקון**
4. **חיבור לאורות העליונים**

### מטרות אישיות

- **זיהוי שורש הנשמה**
- **הבנת תפקיד אישי**
- **פיתוח כלים רוחניים**
- **העמקת הדבקות**

## 📖 סדר הלימוד

### לימוד יומי

3. **ערב**: ספרי חסידות

### לימוד שבועי
