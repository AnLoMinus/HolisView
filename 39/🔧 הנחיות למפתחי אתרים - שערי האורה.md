## 🎨 הנחיות Bootstrap

### 📚 גרסה וספריות

- שימוש ב-Bootstrap 5.3 ומעלה
- שילוב עם RTL Bootstrap
- ספריות נוספות נדרשות:
  - Bootstrap Icons
  - Bootstrap RTL
  - Popper.js

### 🏗️ מבנה Grid

- שימוש במערכת ה-Grid של Bootstrap לעימוד רספונסיבי
- התאמת הקונטיינרים:

  ```html
  <div class="container-fluid" dir="rtl">
  <div class="row">
    <div class="col-md-3">תפריט צד</div>
    <div class="col-md-9">תוכן ראשי</div>
  </div>
  </div>
  ```

### 🧩 רכיבים מומלצים

- Navbar מותאם לניווט RTL
- Cards לתצוגת תוכן מודולרי
- Accordion לתצוגת תוכן מתקפל
- Tabs לניווט בין נושאים
- Modals להצגת מידע נוסף
- Tooltips להסברים קצרים

### 📱 קלאסים רספונסיביים

- שימוש ב-breakpoints:
  - sm: 576px
  - md: 768px
  - lg: 992px
  - xl: 1200px
  - xxl: 1400px

### 🎨 התאמות עיצוב

- שימוש במשתני SCSS מותאמים:

  ```scss
  $primary: #1E3D59;
  $secondary: #FFD700;
  $font-family-base: "Arial", "Narkisim", sans-serif;
  ```

### ⚙️ JavaScript Plugins

- הפעלת פלאגינים נדרשים:

  ```javascript
  // Enable tooltips
  const tooltipTriggerList = document.querySelectorAll('[data-bs-toggle="tooltip"]')
  const tooltipList = [...tooltipTriggerList].map(tooltipTriggerEl => new bootstrap.Tooltip(tooltipTriggerEl))

  // Enable popovers
  const popoverTriggerList = document.querySelectorAll('[data-bs-toggle="popover"]')
  const popoverList = [...popoverTriggerList].map(popoverTriggerEl => new bootstrap.Popover(popoverTriggerEl))
  ```

### 🔧 התאמות RTL

- הוספת תמיכה ב-RTL:

  ```html
  <html dir="rtl" lang="he">
  <head>
    <link rel="stylesheet" href="bootstrap.rtl.min.css">
  </head>
  ```
